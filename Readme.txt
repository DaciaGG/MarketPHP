SupermarketProject


Personal project using MVC patern of a supermarket where the user can sing in, log in (with the possibility of being admin if needed). The user can create a category or a product to be add in the database and editing it. The product are listed in the main aand the differents categories' pages. The possibility to use a cart is present, too. The user can add a product and modify the quantity, modifyng also the total price. After confirmation, the adress is needed to create the order and after the payment, the order is confirm and ready to be treated.

Refers to "Chart.png" and "homePage.png" for preview.

Langages: html, css, php, sql.
